# 🚀 Phase 6 – Spark Playground Exit Sprint

## 📌 Overview

This phase focuses on strengthening PySpark proficiency through intensive practice on joins, window functions, date operations, and pipeline execution.
It serves as a final preparation step before working fully in Databricks environments.

---

## 🎯 Objective

* Build fluency in PySpark transformations
* Practice joins and window functions extensively
* Perform date-based analysis
* Execute a complete data pipeline under time constraints

---

## 🛠️ Practice Areas

### 🔹 Join Operations

* Inner join for valid records
* Left join to identify missing data
* Left anti join to detect invalid foreign keys
* Row count comparisons across joins

---

### 🔹 Window Functions

* Top customers per city using ranking
* Running total of sales
* Ranking customers by spend
* Using LAG to analyze previous records

---

### 🔹 Date Analysis

* Extracting month from date columns
* Monthly sales aggregation
* Calculating date differences
* Trend analysis over time

---

### 🔹 Timed Pipeline Execution

* Data ingestion
* Cleaning and validation
* Referential integrity checks
* Joins and transformations
* Aggregations and window functions
* Output generation within time constraints

---

* Gained confidence in building pipelines independently
