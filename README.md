# 🚀 Data Engineering Training Repository (Capgemini Program)

## 📌 Overview

This repository documents my structured learning journey through the Capgemini Data Engineering training program.
It showcases hands-on practice, project implementations, and progressive skill development across SQL, PySpark, and modern data engineering workflows.

The repository is designed to reflect a transition from foundational concepts to building end-to-end data pipelines and business analytics solutions.

---

## 🎯 Purpose

* Demonstrate practical understanding of data engineering concepts
* Showcase hands-on implementation using SQL and PySpark
* Track learning progress through structured phases
* Build a portfolio of real-world data processing workflows

---

## 🗂️ Repository Structure

### 📁 Phase 0 – Databricks Learning Pathway

* Completed foundational modules on:

  * Data ingestion (Lakeflow Connect)
  * Workflow orchestration (Lakeflow Jobs)
  * Pipeline development (Spark Declarative Pipelines)
  * DevOps fundamentals
* Includes certification badges and summary documentation

* Working with real datasets

### 📁 Phase 3 – ETL Pipeline Development
